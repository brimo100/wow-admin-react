<h1 align="center">wow-admin-react</h1>
<div align="center">
    A simple template of react.
</div>

## support
- React v18
- React-Router v6
- Mobx6
- TypeScript
- Vite 3
- SCSS
- Antd
- Axios
- 集成 Eslint + Standard  来规范和格式化代码
- 集成 Mock 辅助开发
- 国际化：`i18next` `react-i18next`

## usage

```bash
git clone https://github.com/ueumd/wow-admin-react.git

## dev
pnpm i
pnpm dev

## prod
pnpm run build
cd dist
http-server
```
